# Bigballs

Tienes que empacar el juego tú mismo **Bigballs**  
No te ralles, es fácil

---

## Descarga data

Descarga la carpeta `data` desde este enlace:  
[data.zip](https://www.mediafire.com/file/7ulttrepmmt96hw/data.zip/file)

Después, **descomprime `data`**.

Asegurate de tener una carpeta "data" con todo el contenido que has descomprimido.

---

## Agrega las carpetas del repositorio

Descarga de este repositorio de GitHub los archivos:  

- `locales.zip`  
- `resources.zip`  

Descomprime estos dos archivos para tener las carpetas "locales" y "resources" por separado.

Luego, mete estas dos carpetas dentro de `data`.

---
